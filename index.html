HTML Semantic Reference

Semantic Tags

Tag Name Description
<header> Header This element is a container usually for either navigational links or introductory content containing <h1> to <h6> headings.
<nav> Navigation This element is used to define a block of navigation links such as menus and tables of contents.
<main> Main This element is used to encapsulate the dominant content within a webpage.
<footer> Footer The content at the bottom of the subject information is known as the footer, indicated by the <footer> element.
<section> Section This element defines elements in a document, such as chapters, headings, or any other area of the document with the same theme.
<article> Article This element holds content that makes sense on its own.
<aside> Aside This element is used to mark additional information that can enhance another element but isnâ€™t required in order to understand the main content.
<figure> Figure This is an element used to encapsulate media such as an image, illustration, diagram, code snippet, etc, which is referenced in the main flow of the document.
<figcaption> Figure Caption This is an element used to describe the media in the <figure> tag
